# Tutorial-on-GRAD-CAM-for-the-MNIST-dataset
This repository provides a simple and modular implementation of Grad-CAM for CNN image classifiers. The underlying architecture is deliberately minimalistic to emphasize the principles and mechanics of the Grad-CAM explainability method.

The implementation is designed to be easily reusable and extensible, allowing it to be integrated into others pedagogical/ small Proof of Concept computer vision project.

Beyond explainability, this work aims to investigate privacy-preserving model interpretation. Specifically, the ultimate goal is to demonstrate the feasibility of performing Grad-CAM computations over encrypted data using the CKKS homomorphic encryption scheme.
